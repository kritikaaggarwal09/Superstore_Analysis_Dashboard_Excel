# Superstore_Analysis_Dashboard_Excel

## 📌 Project Overview

This project is a *Store Sales Analytics Dashboard* created using *Microsoft Excel*.  
The dashboard helps analyze and visualize store sales data to uncover valuable business insights such as sales trends, profit performance, regional analysis, and product category performance.

The main objective of this project is to transform raw sales data into an interactive and easy-to-understand dashboard for better business decision-making.

---

## 🚀 Features

- Interactive Excel Dashboard
- Data Cleaning & Transformation
- Sales Performance Analysis
- Profit & Revenue Tracking
- Total Quantity Sold
- Region-wise Sales Analysis
- Product Category Insights
- Monthly & Yearly Sales Trends
- User-friendly Visualization
- Dynamic Filters & Slicers
 
  --- 

## 🛠️ Tools & Technologies Used

- Pivot Tables

- Cleaned and processed data to ensure accuracy and consistency.
